# Proyecto Teoría de Grafos

Implementación de algoritmos clásicos de teoría de grafos en Python.

## Algoritmos implementados
- Havel-Hakimi
- Kruskal
- Prim

## Herramientas
- Python
- NetworkX
- Matplotlib

## Objetivos
- Comprender el funcionamiento interno de los algoritmos.
- Analizar su complejidad temporal.
- Visualizar los resultados sobre grafos generados automáticamente.
